# 깃허브 내에 있는 폴더별 설명입니다.

※ Stock_Trading_System -> 프로젝트 실행 코드가 담긴 폴더입니다.
    * 32bit 폴더 설명
      - 32bit 폴더는 32bit 가상환경에서 실행해야 하는 소스코드가 들어 있는 폴더입니다. 대신증권 API에서 주가 데이터를 가져오기 위해서는 32bit 환경이 필요합니다.
        └ "daystockdata"    : 대신증권 API에서 가져온 일 단위 데이터(1개)를 저장해 두는 csv파일이 들어있는 폴더입니다.
        └ "hapdaystockdata" : 대신증권 API에서 가져온 일 단위 데이터(1년치 데이터 + 어제 날짜 기준으로 가져온 일 단위 데이터)를 저장해 두는 csv파일이 들어있는 폴더입니다.
        └ "hapminstockdata" : 대신증권 API에서 가져온 분 단위 데이터(1년치 데이터 + 실시간으로 가져온 분 단위 데이터)를 저장해 두는 csv파일이 들어있는 폴더입니다.
        └ "hapnewsdata"     : 네이버 증권 뉴스에서 가져온 뉴스 데이터(모든 날짜의 뉴스)를 저장해 두는 csv파일이 들어있는 폴더입니다.
        └ "jipyo"           : 일 단위 데이터를 가져다가 지표로 만든 지표 데이터를 저장해 두는 csv파일이 들어있는 폴더입니다.
        └ "minstockdata"    : 대신증권 API에서 가져온 분 단위 데이터(실시간으로 가져온 분 단위 데이터 1개)를 저장해 두는 csv파일이 들어있는 폴더입니다.
        └ "newsdata"        : 네이버 증권 뉴스에서 가져온 뉴스 데이터(뉴스 1~2페이지 - 최신 뉴스만)를 저장해 두는 csv파일이 들어있는 폴더입니다.

    * 소스코드 파일 설명
      - 각 .py 파일별 간략 설명입니다.
        └ "daystock.py"            : 대신증권 API에서 실시간 일 단위 데이터를 가져오는 코드입니다. 가져온 데이터를 각각 "hapminstockdata", "minstockdata" 안에 있는 csv 파일에 최신화 하는 코드입니다.
        └ "filterstockcodename.py" : 거래 가능한 주식종목중에서 '루닛' 종목코드를 가져오는 코드입니다.
        └ "jipyo.py"               : 일 단위 데이터(1년치 + 어제 날짜)를 가져와 지표 데이터로 만드는 코드입니다.
        └ "main.py"                : 모든 .py 코드 파일들을 실행하는 메인 코드입니다. -> 프로젝트 실행시 이 코드를 실행하세요.
        └ "minstock.py"            : 대신증권 API에서 실시간 분 단위 데이터를 가져오는 코드입니다. 가져온 데이터를 각각 "hapdaystockdata", "daystockdata" 안에 있는 csv 파일에 최신화 하는 코드입니다.
        └ "news.py"                : 당일 뉴스 데이터를 웹크롤링으로 가져와 "hapnewsdata", "newsdata" 안에 있는 csv 파일에 최신화 하는 코드입니다.
        └ "stockapicheck.py"       : 대신증권 API와 연결이 잘 되었는지 체크하는 코드입니다.
        └ "stockHTS.py"            : 강화학습 신경망에서 나온 결과를 토대로 실시간 주식 매수, 매도, 홀드를 하고 결과를 DB에 올리는 코드입니다.
        └ "stocknewsdataupload.py" : 실시간 분, 일 단위 주가 데이터와 실시간으로 올라온 뉴스 데이터를 DB에 업로드 하는 코드입니다.
        └ "subpub32bit.py"         : 대신증권 API에서 실시간 주가 데이터를 가져와 DB에 업로드 했다는 메세지를 64bit 환경에 전송하는 코드입니다.
        └ "timeCh.py"              : 현재 시간을 원하는 형식으로 바꿔주는 코드입니다.



    * 64bit 폴더 설명
      - 64bit 폴더는 64bit 가상환경에서 실행해야 하는 소스코드가 들어 있는 폴더입니다. 딥러닝(pytorch, tensorflow 등등) 해당 모듈 설치가 64bit 환경에서만 설치가 가능합니다.

    * 소스코드 파일 설명
      - 각 .py 파일별 간략 설명입니다.
        └ "databased.py"             : DB에 들어있는 일, 분 단위 데이터와 실시간 뉴스 데이터를 가져오는 코드입니다.
        └ "datahap.py"               : 실시간 분 단위 데이터와 지표 데이터, 실시간 뉴스 라벨링 데이터를 하나로 합치는 코드입니다.
        └ "jipyo.py"                 : 일 단위 데이터(1년치 데이터 + 어제 날짜 기준으로 가져온 일 단위 데이터)를 전부 가져와 지표 데이터로 만드는 코드입니다.
        └ "mainLinux.py"             : 모든 .py 코드 파일들을 실행하는 메인 코드입니다. -> 프로젝트 실행시 이 코드를 실행하세요.
        └ "newslabel.py"             : 실시간 뉴스 내용을 긍정, 부정으로 체점하는 코드입니다. -> 자연어 처리
        └ "reinforcementLearning.py" : 실시간 데이터를 LSTM 모델에 predict하는 코드입니다.
        └ "subpub64bit.py"           : 32bit 환경으로부터 메세지를 수신 받고, 주식 매수, 매도, 홀드 명령을 32bit로 회신하는 코드입니다.
        └ "timeCh.py"                : 현재 시간을 원하는 형식으로 바꿔주는 코드입니다.

    * 64bit 폴더내 파일 설명
      - "lstm.hdf5" 파일은 2년치 데이터를 학습시킨 모델입니다. 해당 모델에 실시간 주식 데이터를 predict시, 매수, 매도, 홀드 action 예측값이 출력됩니다.



※ 사진 및 PPT 자료 -> 프로젝트를 진행하면서 캡쳐한 사진 또는 PPT제작에 사용된 사진 파일들이 있습니다.

※ 설치 파일
    * 설치 폴더내 파일 설명
      - "Redis-x64-3.0.504.msi" 파일은 redis를 사용하기 위해 윈도우 운영체제에 설치하는 파일입니다.
      - "TA_Lib-0.4.24-cp310-cp310-win32.whl" 파일은 32bit 환경에 Talib을 설치할 때 필요한 파일입니다.(파이썬 3.10.x 버전)
      - "TA_Lib-0.4.24-cp310-cp310-win_amd64.whl" 파일은 64bit 환경에 Talib을 설치할 때 필요한 파일입니다.(파이썬 3.10.x 버전)
      - "triton-2.0.0-cp310-cp310-win_amd64.whl" 파일은 64bit 환경에 triton 모듈을 설치할때 필요한 파일입니다. -> 오류가 발생할 시 설치하세요.
